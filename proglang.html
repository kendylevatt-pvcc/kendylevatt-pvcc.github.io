Chapter 6: Programming Language

Kendyl Evatt

Term: Definition:
Machine language Language understood by computer programming
Debugging The process of finding and fixing errors in code
Assembler A program for converting instructions into machine code
Assembly language A programming language to communicate with a computer's hardware
Translator A program that translates to another program
Compiler Translates code in one program to another
Interpreter A program that executes instructions without converting into code
Cross-platform software Software that can run on multiple operating systems
Natural language A method of writing software that uses everyday language
Formal language A system of symbols and rules to define computer programming
Machine Independence A program that can run on different computers and equipment
High-level programming language Examples: Python, JavaScript, Visual Basic, Delphi, Perl
Object-oriented programming (OOP) Organizes software around objects or data
Scripting language Interprets instructions individually at runtime
Data types - Integer Whole numbers used to represent numerical value
Float A number positive or negative containing one or more decimals
String Data that stores a sequence of symbols, characters, or words
Primitive data type Data types that specify the size and type of variables
Data structure How data is stored and organized
Python collection data structures - List Written in square brackets; can be ordered and changed
Tuple Written in round brackets; can be ordered but is unchangeable
Set A collection that is unordered, unchangeable, and unindexed
Dictionary A collection that is unordered, changeable, and indexed
Function Defined using the def keyword
Local variable Only available within a specific function or block of code
Global variable A variable that can be accessed from any part of a program
Source program An original computer program that is converted into the equivalent object program
Object program A program translated by the compiler or assembler
