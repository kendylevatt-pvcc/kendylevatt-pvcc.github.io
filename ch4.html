Term Definition
LAN Local Area Network
MAN Metropolitan Area Network
WAN Wide Area Network
Bus Network all devices are connected to a single cable and can commuicate with others
Star Network A network topology where all devices are connected to a central hub or switch that acts as a network controller
CSMA/CD Ethernet Used in Ethernet networds to prevent collisions when transiting multiple data sources at the same time
CSMA/CA Wifi Protocol used in wireless networks to reduce risks of collision
Repeater passes signals back and forth between two original buses
bridge looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side
switch a bridge with multiple connections, allowing it to connect several buses rather than just two thus, a switch produces a network consisting of several buses extending from the switch, as spokes on a wheel (
router provide links between networks while allowing each network to maintain its unique internal characteristics
gateway The “point” at which one network is linked to an internet
P2P: Peer-to-Peer the client/server model involves one process (the server) providing a service to numerous others (clients)
Distributing system distributes service tasks over multiple peers
Cloud Computing delivery of computing resources over the internet
Tier 1 largest most powerful tier with global access and transmitable data
Tier 2 A regional tier tjat peers with many networks but may include transit fees
Tier 3 A tier that must pay fees in order to access the internet
ICANN ICANN
TLD Top Level Domain
Country-Code TLDs TLDs
FTP File Transfer Protocol
HTTP Hypertext Transfer Protocol
SMTP Simple Mail Transfer Protocol
URL Uniform Resource Locator
Client-service activities client server model
Server-side activities secirity and developmental services
application layer top layer of the OSI model of the internet
transport layer manages the end-to-end communication between devices
network layer facilitates data between networks
link layer manages transmission of data between a network segment
packet-switching newtwork data transmission over a network by breaking it into smaller packets and sent individually
fault-tolerant a systems ability to keep operating when components fail
redundant technique that creates multiple points of data to flow increasing availibility
malware attacks that incorporate malicious software
virus software that infects a computer by inserting itself into programs that already reside in the machine
worm autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers
trojan horse a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim
spyware software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack
phishing a technique of obtaining information explicitly by simply asking for it. The term phishing involves numerous “lines” in hopes that someone will “take the bait
DoS attack which is the process of overloading a computer with messages
firewall designed to block outgoing messages with certain destination addresses or to block incoming messages from origins that are known to be sources of trouble
spam filter distingusihes between email and spam mail
proxy server a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server
antivirus software detects and remove the presence of known viruses and other infections
cryptography the study of sending and receiving messages securely in the presence of adversaries. Cryptographic tools such as encryption can be used to transform a message into a coded version such that even if the encrypted data fall into unscrupulous hands, the encoded information will remain confidential
public key encryption a 2 key system used to encrypt messages and the other key is required to decrypt messages
